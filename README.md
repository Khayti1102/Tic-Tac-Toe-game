# Tic-Tac-Toe-game
A simple web-based implementation of the classic Tic-Tac-Toe Game using HTML, CSS, JavaScript.

## Check live preview here :
[Click Here](https://khayti1102.github.io/Tic-Tac-Toe-game/)

##Teach Stack
HTML, CSS, JavaScript
 
##Features
* [ ] 3x3 Grid Board
* [ ] Two-Player Mode: taking turns ("X" and "O").
* [ ] Turn Management
* [ ] Move Validation
* [ ] Draw Detection
* [ ] Draw Detection
* [ ] Game Status Display
* [ ] Reset Functionality
